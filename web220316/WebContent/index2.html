1.블록 레벨 텍스트 태그

태그 혼자서 한 줄을 차지하며 너비가 100%로 꽉 차게 표현되는 태그

h태그: h1 >h2 >h3 >h4 >h5 >h6 -heading

p태그: 단락을 표현하는 태그

hr태그: 주제가 바뀌거나 분위기 전환을 할때 사용하는 가로선

2.인라인 레벨 텍스트 태그

컨텐츠 만큼의 너비를 차지하는 태그

strong태그: 텍스트를 굵게 표시하는 태그 span태그: 텍스트의 일부만 특정 스타일처리를 한다던가 할때 사용

3.목록을 만드는 태그

ul태그(unordered list):순서없는 목록

  • 메뉴1
  • 메뉴2
  • 메뉴3

ul의 list-style-type이 기본 disc이다


ol태그(ordered list):순서있는 목록

  1. 메뉴1
  2. 메뉴2
  3. 메뉴3

ol의 list-style-type이 기본 decimal(정수)이다.
숫자, 알파벳, 로마숫자로 순서를 표시한다. 설정변경: type
시작값을 지정가능: start

4.표를 만드는 태그

table태그, tr(table row)태그, td(table data)태그
행과 열로 이루어진 표의 형태를 만들때 사용하는 태그
tr: 행에 해당하는 태그
td: 행 내부의 열에 해당하는 셀 태그
th: table head 행 내부의 제목에 해당하는 태그
